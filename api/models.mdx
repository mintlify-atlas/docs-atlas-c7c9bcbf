## Overview

The `ModelsResource` provides methods to fetch detailed information about models and their generation parameters.

Accessible via `client.models` (sync) or `client.models` (async).

## Methods

### detail()

Fetch detailed information about a model, including metadata, tags, statistics, and sample images.

<ParamField path="model_no" type="str" required>
  Model identifier to look up
</ParamField>

<ParamField path="scene" type="str" default="'model_detail_v2'">
  Scene context forwarded to the API
</ParamField>

<ResponseField name="return" type="APIEnvelope[ModelDetails]">
  API envelope containing comprehensive model information
  
  <Expandable title="ModelDetails fields">
    <ResponseField name="id" type="str">
      Model identifier
    </ResponseField>
    
    <ResponseField name="name" type="str">
      Model name
    </ResponseField>
    
    <ResponseField name="desc" type="str">
      Model description
    </ResponseField>
    
    <ResponseField name="author" type="Author">
      Author information
    </ResponseField>
    
    <ResponseField name="base_model" type="str">
      Base model name (e.g., "SDXL", "SD1.5")
    </ResponseField>
    
    <ResponseField name="model_ver_no" type="str">
      Current model version identifier
    </ResponseField>
    
    <ResponseField name="version_name" type="str">
      Version name
    </ResponseField>
    
    <ResponseField name="cover" type="str">
      Cover image URL
    </ResponseField>
    
    <ResponseField name="samples" type="list[Sample]">
      Sample images generated with this model
    </ResponseField>
    
    <ResponseField name="tags" type="list[Tag]">
      Model tags for categorization
    </ResponseField>
    
    <ResponseField name="stat" type="Stat">
      Statistics (num_of_artwork, rating, num_of_like, num_of_collection, num_of_task, num_of_view)
    </ResponseField>
    
    <ResponseField name="model_ver_info_v2" type="ModelVerInfoV2">
      Version-specific information including trigger words and recommended generation parameters
    </ResponseField>
    
    <ResponseField name="trained_words" type="list[str] | None">
      Trigger words for LoRA models
    </ResponseField>
    
    <ResponseField name="create_at" type="int">
      Creation timestamp
    </ResponseField>
    
    <ResponseField name="publish_status" type="int">
      Publication status code
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```python Sync
from seaart import SyncClient

with SyncClient() as client:
    client.auth.login(email="user@example.com", password="password")
    
    # Get model details
    info = client.models.detail("model_abc123")
    
    # Access model information
    print(f"Name: {info.value.name}")
    print(f"Base model: {info.value.base_model}")
    print(f"Rating: {info.value.stat.rating}")
    print(f"Total artworks: {info.value.stat.num_of_artwork}")
    
    # Check trigger words for LoRA models
    if info.value.trained_words:
        print(f"Trigger words: {', '.join(info.value.trained_words)}")
    
    # View samples
    for sample in info.value.samples:
        print(f"Sample: {sample.url}")
```

```python Async
from seaart import AsyncClient

async with AsyncClient() as client:
    await client.auth.login(email="user@example.com", password="password")
    
    # Get model details
    info = await client.models.detail("model_abc123")
    
    print(f"Name: {info.value.name}")
    print(f"Base model: {info.value.base_model}")
```
</CodeGroup>

### generation_parameters()

Fetch generation parameters for a specific model version. This provides the parameter panels (basic, advanced, image generation) with type information, limits, and default values.

<ParamField path="model_no" type="str" required>
  Model identifier
</ParamField>

<ParamField path="model_ver_no" type="str" required>
  Model version identifier
</ParamField>

<ParamField path="base_model_name" type="str | None" default="None">
  Optional base model name to filter parameters
</ParamField>

<ResponseField name="return" type="APIEnvelope[GenerationParameters]">
  API envelope containing generation parameter configuration
  
  <Expandable title="GenerationParameters fields">
    <ResponseField name="model_type" type="str | None">
      Model type (e.g., "txt2img", "img2img")
    </ResponseField>
    
    <ResponseField name="max_niter" type="int | None">
      Maximum number of iterations
    </ResponseField>
    
    <ResponseField name="img_param" type="ImgParam | None">
      Image generation parameters containing:
      - `basic_panel` - Basic parameters (resolution, aspect ratio, etc.)
      - `advanced_panel` - Advanced parameters (steps, CFG scale, sampler, etc.)
      - `image_gen_param_panel` - Additional generation parameters
      - `additional_model_panel` - Additional model-specific parameters
    </ResponseField>
    
    <ResponseField name="t2v_param" type="Any | None">
      Text-to-video parameters (if applicable)
    </ResponseField>
    
    <ResponseField name="i2v_param" type="Any | None">
      Image-to-video parameters (if applicable)
    </ResponseField>
    
    <ResponseField name="v2v_param" type="Any | None">
      Video-to-video parameters (if applicable)
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```python Sync
from seaart import SyncClient

with SyncClient() as client:
    client.auth.login(email="user@example.com", password="password")
    
    # Get generation parameters
    params = client.models.generation_parameters(
        model_no="model_abc123",
        model_ver_no="ver_xyz"
    )
    
    # Access parameter information
    print(f"Model type: {params.value.model_type}")
    print(f"Max iterations: {params.value.max_niter}")
    
    # Get resolution for aspect ratio
    if params.value.img_param:
        size = params.value.get_size("16:9")
        if size:
            width, height = size
            print(f"16:9 resolution: {width}x{height}")
    
    # Access basic panel parameters
    if params.value.img_param:
        for block in params.value.img_param.basic_panel.component_block_items:
            print(f"Component: {block.component_type}")
            if block.resolution_param:
                print("  Resolution options available")
```

```python Async
from seaart import AsyncClient

async with AsyncClient() as client:
    await client.auth.login(email="user@example.com", password="password")
    
    # Get generation parameters
    params = await client.models.generation_parameters(
        model_no="model_abc123",
        model_ver_no="ver_xyz"
    )
    
    print(f"Model type: {params.value.model_type}")
```
</CodeGroup>

## Helper Methods

### get_size()

The `GenerationParameters` object includes a helper method to get resolution dimensions for a given aspect ratio.

```python
params = client.models.generation_parameters("model_no", "ver_no")

# Get resolution for aspect ratio
size = params.value.get_size("16:9")
if size:
    width, height = size
    print(f"Resolution: {width}x{height}")
```

Supported aspect ratios:
- `"1:1"` - Square (1024x1024)
- `"16:9"` - Widescreen (1024x592)
- `"9:16"` - Portrait mobile (592x1024)
- `"4:3"` - Standard (1024x768)
- `"3:4"` - Portrait (768x1024)
- `"21:9"` - Ultrawide
- And more...

## Usage Examples

### Finding Model Information

```python
from seaart import SyncClient

with SyncClient() as client:
    client.auth.login(email="user@example.com", password="password")
    
    # Search for a model
    search_result = client.search.models("anime style", order_by="hot")
    
    if search_result.value.items:
        model_id = search_result.value.items[0].id
        
        # Get detailed information
        model = client.models.detail(model_id)
        
        print(f"Model: {model.value.name}")
        print(f"Author: {model.value.author.name}")
        print(f"Rating: {model.value.stat.rating}")
        print(f"Base: {model.value.base_model}")
        
        # Get generation parameters
        params = client.models.generation_parameters(
            model_no=model_id,
            model_ver_no=model.value.model_ver_no
        )
        
        # Get recommended resolution
        size = params.value.get_size("1:1")
        if size:
            print(f"Recommended size: {size[0]}x{size[1]}")
```

### Using Trigger Words

```python
# Get model details
model = client.models.detail("lora_model_id")

# Check for trigger words
if model.value.trained_words:
    trigger_words = ", ".join(model.value.trained_words)
    prompt = f"{trigger_words}, a beautiful landscape"
    print(f"Using prompt with triggers: {prompt}")
else:
    prompt = "a beautiful landscape"
    print(f"No trigger words needed: {prompt}")
```
