## Exception Hierarchy

All SeaArt exceptions inherit from `SeaArtError`, which extends Python's base `Exception` class.

```
Exception
└── SeaArtError
    ├── APIError
    │   ├── AuthError
    │   │   ├── AccountNotFoundError
    │   │   ├── InvalidPasswordError
    │   │   ├── AuthTokenInvalidError
    │   │   ├── AccountNotLoggedInError
    │   │   └── AccountBlockedError
    │   ├── AccountForbiddenError
    │   ├── AccountCancelOncePerDayError
    │   ├── TaskCompletionCannotBeCanceledError
    │   ├── ParamsError
    │   ├── ServerError
    │   ├── ArtModelNoIsEmptyError
    │   ├── ComfyuiApiNodeCannotBeCanceledError
    │   ├── TaskHangLimitExceededError
    │   ├── UsernameError
    │   ├── TouristChatNumLimitError
    │   ├── PromptHaveSensitiveWordsError
    │   ├── FeatureRequiresPaymentError
    │   ├── ImageCountExceedLimitError
    │   ├── DoNotTurnOffPermissionToPublishWorksToTheCommunityError
    │   ├── BalanceNotEnoughError
    │   ├── OperationNotAllowedError
    │   ├── ProcessingError
    │   └── AccountNotMatchError
    ├── MissingDataError
    ├── TaskTimeoutError (also inherits from TimeoutError)
    ├── InvalidRequestError (also inherits from ValueError)
    └── StreamInterruptedError
```

## Base Exceptions

### SeaArtError

```python
class SeaArtError(Exception):
    """Base exception for all SeaArt client errors."""
```

Base class for all exceptions raised by the SeaArt SDK. Catch this to handle any SeaArt-related error.

**Example:**
```python
from seaart import SeaArt, SeaArtError

client = SeaArt()

try:
    await client.characters.chat(character_id, "Hello")
except SeaArtError as e:
    print(f"SeaArt error: {e}")
```

---

### APIError

```python
class APIError(SeaArtError):
    """Generic API error raised for HTTP or logic failures."""
    
    def __init__(
        self,
        status: int,
        body: bytes,
        *,
        code: int | None = None,
        msg: str | None = None,
    )
```

Generic API error raised for HTTP or logic failures.

**Attributes:**
- `http_status` (int): Raw HTTP status code
- `code` (int | None): Optional API error code
- `msg` (str | None): Optional message returned by the API
- `body` (bytes): Raw response body as bytes

**Example:**
```python
from seaart import SeaArt, APIError

try:
    await client.images.create(prompt="test")
except APIError as e:
    print(f"HTTP {e.http_status}")
    print(f"Error code: {e.code}")
    print(f"Message: {e.msg}")
```


## Server & Processing Errors

### ServerError

```python
class ServerError(APIError):
    """Raised when the server encounters an internal error."""
```

**When raised:** When the SeaArt API experiences an internal server error.

**Example:**
```python
from seaart import SeaArt, ServerError
import asyncio

async def retry_on_server_error():
    max_retries = 3
    for attempt in range(max_retries):
        try:
            return await client.images.create(prompt="test")
        except ServerError:
            if attempt < max_retries - 1:
                await asyncio.sleep(2 ** attempt)  # Exponential backoff
            else:
                raise
```

### StreamInterruptedError

```python
class StreamInterruptedError(SeaArtError):
    """Raised when a streaming response ends without receiving an 'end' event.
    
    This typically indicates a network interruption or an unexpected server-side
    termination of the stream before the final event was delivered.
    """
```

**When raised:** During streaming operations when the connection is interrupted.

**Example:**
```python

        print(chunk, end="", flush=True)
except StreamInterruptedError:
    print("\nStream interrupted. Please try again.")
```

## Error Handling Best Practices

### Basic Error Handling

```python
from seaart import SeaArt, SeaArtError, APIError

client = SeaArt()

try:
    result = await client.images.create(prompt="A beautiful landscape")
except SeaArtError as e:
    print(f"Error: {e}")
```

### Specific Error Handling

```python
from seaart import (
    SeaArt,
    BalanceNotEnoughError,
    PromptHaveSensitiveWordsError,
    TaskTimeoutError,
    ServerError,
)

try:
    result = await client.images.create_and_wait(
        prompt="A serene mountain landscape",
        timeout=60
    )
except BalanceNotEnoughError:
    print("Please add more credits to your account")
except PromptHaveSensitiveWordsError:
    print("Prompt contains sensitive content. Please revise.")
except TaskTimeoutError:
    print("Image generation took too long. Please try again.")
except ServerError:
    print("Server error. Please try again later.")
```

### Authentication Error Handling

```python
from seaart import (
    SeaArt,
    AccountNotFoundError,
    InvalidPasswordError,
    AuthTokenInvalidError,
)

async def login_with_retry(email: str, password: str):
    client = SeaArt()
    
    try:
        await client.auth.login(email=email, password=password)
    except AccountNotFoundError:
        print("Account not found. Please sign up first.")
        return None
    except InvalidPasswordError:
        print("Invalid password. Please try again.")
        return None
    except AuthTokenInvalidError:
        print("Authentication failed. Please contact support.")
        return None
    
    return client
```

### Retry Logic with Exponential Backoff

```python
import asyncio
from seaart import SeaArt, ServerError, TaskTimeoutError

async def create_image_with_retry(prompt: str, max_retries: int = 3):
    client = SeaArt()
    
    for attempt in range(max_retries):
        try:
            return await client.images.create_and_wait(prompt=prompt)
        except ServerError:
            if attempt < max_retries - 1:
                wait_time = 2 ** attempt  # Exponential backoff: 1, 2, 4 seconds
                print(f"Server error. Retrying in {wait_time}s...")
                await asyncio.sleep(wait_time)
            else:
                print("Max retries reached. Please try again later.")
                raise
        except TaskTimeoutError:
            print("Task timed out.")
            raise
```
