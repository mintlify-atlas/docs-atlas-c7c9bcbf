# Characters

The Characters resource provides access to AI character chat functionality, including session management, message streaming, and character information.

**Access:** `client.characters`

## Sub-resources

- **`chats`** - Chat sessions and message streaming
- **`settings`** - Global account-level character settings
- **`datacards`** - User persona profiles attached to chats
- **`replies`** - AI-suggested reply recommendations

## Character Methods

### follow

Follow (collect) a character.

```python
from seaart import SyncClient

client = SyncClient()
client.auth.login("user@example.com", "password")

result = client.characters.follow("char_abc123")
follow_id = result.value.id  # Save this for unfollowing later
```

<ParamField path="character_id" type="str" required>
  ID of the character to follow
</ParamField>

<ParamField path="collection_no" type="str | None" default="None">
  Optional collection to add the character to
</ParamField>

<ResponseField name="return" type="APIEnvelope[CollectResponse]">
  Response containing:
  - `.value.id` - Follow-record ID needed to unfollow later
</ResponseField>

### stream

Stream a message to the character and receive chunks in real time.

```python
for item in client.characters.chats.stream(
    content="Hello!",
    session_id=session_id
):
    if item.event == "message":
        print(item.text, end="", flush=True)
    elif item.event == "end":
        print(f"\n\nMessage ID: {item.msg_id}")
```

  Message text to send
</ParamField>

<ParamField path="session_id" type="str" required>

</ParamField>

  Optional character name override
</ParamField>

</ParamField>

</ParamField>
