# Images

The Images resource provides image generation and manipulation operations including text-to-image, image-to-image, upscaling, and background removal.

**Access:** `client.images`

## Sub-resources

- **`history`** - Image usage history

## Generation Methods

### text_to_img

Submit a text-to-image generation task.

```python
from seaart import SyncClient, ImageSize

client = SyncClient()
client.auth.login("user@example.com", "password")

envelope = client.images.text_to_img(
    prompt="a cat in a forest",
    model_no="model_abc123"
)
item = envelope.wait()  # Wait for generation to complete
print(item.image_url)
```

<ParamField path="prompt" type="str" required>
  Text prompt describing the desired image
</ParamField>

<ParamField path="model_no" type="str" required>
  Model identifier to use for generation
</ParamField>

<ParamField path="model_ver_no" type="str | None" default="None">
  Model version identifier. Resolved automatically from cache when `None`
</ParamField>

<ParamField path="gen_mode" type="Literal['standard', 'quality']" default="'standard'">
  Generation quality mode
</ParamField>

<ParamField path="prompt_magic" type="bool | None" default="None">
  Enable AI prompt enhancement before generation. Uses server default when `None`
</ParamField>

<ParamField path="image_size" type="ImageSize" default="ImageSize.PORTRAIT_2_3">
  Output aspect ratio. Use an `ImageSize` preset or `ImageSize.custom()` for exact pixels
</ParamField>

<ParamField path="num_images" type="int" default="1">
  Number of images to generate in one task
</ParamField>

<ParamField path="free_creation" type="bool | None" default="None">
  Use free generation quota when `True`. Uses server default when `None`
</ParamField>

<ParamField path="image_url" type="str | None" default="None">
  Optional reference image URL for conditioning
</ParamField>

<ParamField path="options" type="TextToImgOptions | None" default="None">
  Advanced generation options (negative prompt, sampling parameters, LoRA models, etc.)
</ParamField>

<ResponseField name="return" type="SyncTaskEnvelope[TaskMetadata]">
  Task envelope:
  - `.value.id` - Task ID
  - `.wait()` - Block until generation completes and receive a `TaskItem` with `.image_url` / `.image_urls`
</ResponseField>

## Upscaling Methods

### upscale

Submit an image upscaling task. When `image_url` is a SeaArt CDN URL, model and options are resolved automatically.

```python
# With SeaArt CDN URL (auto-resolves settings)
envelope = client.images.upscale("https://image.cdn2.seaart.me/...")
item = envelope.wait()
print(item.image_url)

# With external URL (must provide model)
envelope = client.images.upscale(
    image_url="https://example.com/image.png",
    model_no="model_abc123",
    options=UpscaleOptions()
)
```

<ParamField path="image_url" type="str" required>
  URL of the image to upscale
</ParamField>

<ParamField path="model_no" type="str | None" default="None">
  Model identifier. Required when URL doesn't contain embedded task ID
</ParamField>

<ParamField path="model_ver_no" type="str | None" default="None">
  Model version identifier. Resolved from cache when `None`
</ParamField>

<ParamField path="prompt" type="str | None" default="None">
  Prompt passed to upscaler. Resolved from original task when `None` and URL contains task ID
</ParamField>

<ParamField path="options" type="UpscaleOptions | None" default="None">
  Advanced upscaling options. Resolved from original task when `None` and URL contains task ID
</ParamField>

<ResponseField name="return" type="SyncTaskEnvelope[TaskMetadata]">
  Task envelope with `.wait()` method
</ResponseField>

**Raises:**
- `InvalidRequestError` - If URL doesn't contain task ID and `model_no` or `options` are not provided

## Upload Methods

### upload

Upload an image file to the SeaArt platform.

```python
# From file path
result = client.images.upload(path="/tmp/photo.png")
print(result.value.url)

# From base64 string
result = client.images.upload(base64_str="iVBORw0KGgoAAAANS...")

# From bytes
with open("/tmp/photo.png", "rb") as f:
    result = client.images.upload(bytes_data=f.read())
```

<ParamField path="path" type="str | Path | None" default="None">
  Filesystem path to the image file
</ParamField>

<ParamField path="base64_str" type="str | None" default="None">
  Base64-encoded image data. Data-URI prefixes are stripped automatically
</ParamField>

<ParamField path="bytes_data" type="bytes | None" default="None">
  Raw image bytes
</ParamField>

<ParamField path="filename" type="str | None" default="None">
  Filename to use in upload. Inferred from `path` when not provided
</ParamField>

<ParamField path="category" type="int" default="36">
  Upload category flag forwarded to the API
</ParamField>

<ResponseField name="return" type="APIEnvelope[UploadedImage]">
  Upload result:
  - `.value.url` - Public image URL
  - `.value.signed` - Signed URL
</ResponseField>

**Raises:**
- `InvalidRequestError` - If not exactly one of `path`, `base64_str`, or `bytes_data` is provided

**Location:** `src/seaart/resources/sync/images.py:649`

<ParamField path="model_ver_no" type="str | None" default="None">
