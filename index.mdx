## Quick Start

Get started with SeaArt SDK in minutes:

<CodeGroup>

```bash pip
pip install seaart
```

```bash uv
uv add seaart
```

</CodeGroup>

```python
from seaart import SyncClient

with SyncClient() as client:
    client.auth.login(email="user@example.com", password="password")
    
    # Chat with characters
    chat = client.characters.chats.create(character_id)
    result = client.characters.chats.send("Hello!", chat.value.session_id)
    print(result.text)
```
